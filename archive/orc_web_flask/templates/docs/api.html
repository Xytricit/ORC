API Documentation

Integrate ORC into your tools and workflows with our REST API.

Authentication

Use your CLI token in the X-CLI-Token header:

curl -H "X-CLI-Token: YOUR_TOKEN" http://127.0.0.1:5000/api/configs

Endpoints

GET /api/configs

Get your API configurations

GET /api/configs
Headers: X-CLI-Token: YOUR_TOKEN

Response:
{
  "configs": [
    {
      "provider": "groq",
      "api_key": "gsk_...",
      "model_name": "llama-3.3-70b-versatile",
      "is_default": true
    }
  ],
  "user": {
    "username": "turbo",
    "email": "user@example.com"
  }
}

GET /api/default-config

Get your default AI provider configuration

GET /api/default-config
Headers: X-CLI-Token: YOUR_TOKEN

Response:
{
  "provider": "groq",
  "api_key": "gsk_...",
  "model_name": "llama-3.3-70b-versatile"
}

POST /api/projects/auto-create

Auto-create a project (used by CLI)

POST /api/projects/auto-create
Headers: X-CLI-Token: YOUR_TOKEN
Body:
{
  "name": "my-project",
  "path": "/path/to/project",
  "db_path": "/path/to/.orc/index.db"
}

Response:
{
  "message": "Project created",
  "project_id": 1,
  "name": "my-project"
}

POST /api/token/generate

Generate a new CLI token (requires web login)

POST /api/token/generate
Cookies: session

Response:
{
  "token": "abcd1234...",
  "created_at": "2026-01-09T14:00:00"
}
