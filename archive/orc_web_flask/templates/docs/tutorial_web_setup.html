Tutorial: Setting Up the Web Interface

Learn how to set up and use ORC's web dashboard.

Step 1: Start the Web Server

The web server should already be running. If not, start it:

python orc/web/app_new.py

Visit: http://127.0.0.1:5000

Step 2: Create an Account

Click Sign Up and create your account with:

  • Username
  • Email
  • Password

Step 3: Configure AI Provider

Go to Settings > API Config and add your AI provider:

  1. Click Add Provider
  2. Choose provider (Groq recommended for free tier)
  3. Enter your API key
  4. Select model
  5. Save and set as default

Step 4: Generate CLI Token

Go to Account page:

  1. Click Generate Token
  2. Copy the token
  3. Use it with orc login --token YOUR_TOKEN

Step 5: Connect CLI

orc login

This will open your browser. Paste the token when prompted.

Step 6: Your First Project

Projects are auto-created when you run orc in a directory!

cd your-project
orc

The project will appear in your web dashboard automatically.

You're All Set!

Now you can:

  • View all projects in the web dashboard
  • Run analyses from the web interface
  • Use AI chat both in CLI and web
  • Track code quality over time
