Web Interface Documentation

ORC's web interface provides a powerful visual way to explore and analyze your codebase.

Dashboard

Your home base showing:

  • All your projects
  • Recent analyses
  • Quick stats

Projects

Manage multiple codebases:

  • Create projects manually or auto-created from CLI
  • View project statistics
  • Run analyses
  • Track changes over time

AI Chat

Interactive chat interface to ask questions about your code in real-time.

Settings

Configure your ORC experience:

  • Account - Generate CLI tokens for authentication
  • API Config - Add AI provider keys (Groq, OpenAI, etc.)

Setting Up API Keys

  1. Go to Settings > API Config
  2. Click Add Provider
  3. Select your AI provider (Groq, OpenAI, etc.)
  4. Enter your API key and model name
  5. Set as default (optional)

Note: API keys configured here are used by the CLI when authenticated.
