CLI Reference

Authentication

orc login

Connect CLI to ORC web application

orc login                    # Interactive login
orc login --token TOKEN      # Direct token login
orc login --apikey KEY       # Direct API key

Analysis Commands

orc

Start AI chat session (requires authentication)

orc

orc index

Index a codebase for analysis

orc index                    # Index current directory
orc index /path/to/code      # Index specific directory
orc index --force            # Force re-indexing

orc analyze

Run full codebase analysis

orc analyze

orc dead

Find potentially unused code

orc dead
orc dead --confidence 0.8    # Set confidence threshold
orc dead --limit 50          # Limit results

orc complexity

Show complexity metrics

orc complexity
orc complexity --threshold 15  # Custom threshold

orc hotspots

Find problem areas

orc hotspots
orc hotspots --limit 20

AI Chat Commands

Inside the ORC chat, use these slash commands:

  • /help - Show all commands
  • /status - Show AI provider status
  • /providers - List available AI providers
  • /models - Show current model
  • /clear - Clear conversation
  • /history - Show chat history
  • /exit - Exit ORC
