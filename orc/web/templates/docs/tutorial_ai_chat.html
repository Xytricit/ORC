Tutorial: Using AI Chat

Learn how to use ORC's AI chat to ask questions about your codebase.

Starting AI Chat

orc login        # Authenticate first
orc              # Start chat

Example Questions

Understanding Code

You: "What does the authentication system do?"

ORC: Analyzes auth-related functions and explains the flow

Finding Issues

You: "Show me the most complex functions"

ORC: Uses get_complexity_report tool and lists results

Code Quality

You: "Are there any unused functions?"

ORC: Runs dead code analysis and reports findings

Refactoring Advice

You: "How can I improve the performance of user_login()?"

ORC: Analyzes the function and suggests optimizations

Useful Slash Commands

  • /models - Change AI model
  • /providers - Switch AI provider
  • /clear - Clear conversation history
  • /status - Check AI provider status
  • /help - Show all commands

Tips for Better Results

  • Be specific: "Show complex functions in auth.py" vs "show me complex stuff"
  • Ask follow-up questions: ORC remembers context from previous messages
  • Use /clear when switching topics to reset context
  • Index your code first with orc index
