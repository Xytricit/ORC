Tutorial: Your First Code Analysis

In this tutorial, you'll learn how to analyze your codebase with ORC for the first time.

Step 1: Navigate to Your Project

cd /path/to/your/project

Make sure you're in the root directory of the codebase you want to analyze.

Step 2: Initialize ORC (Optional)

orc init

This creates configuration files. Skip if you want to use defaults.

Step 3: Index Your Code

orc index

ORC will scan your codebase and create a searchable database. This may take a minute for large projects.

What's happening: ORC is parsing your code, extracting functions, classes, and dependencies.

Step 4: View Statistics

orc stats

See a summary of your codebase: file counts, function counts, language breakdown.

Step 5: Run Analysis

orc analyze

Get insights about complexity, large files, and potential issues.

Step 6: Find Dead Code

orc dead

Discover functions that might not be used anywhere in your codebase.

Congratulations!

You've completed your first ORC analysis! Here's what to try next:

  • Check out the web interface at http://127.0.0.1:5000
