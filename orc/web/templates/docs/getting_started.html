Getting Started with ORC

Welcome! Let's get you up and running with ORC in just a few minutes.

Installation

Step 1: Install ORC

pip install orc-cli

Or install from source:

git clone https://github.com/yourusername/orc.git
cd orc
pip install -e .

Authentication

Step 2: Login to ORC

Two authentication methods available:

Method 1: Web Token (Recommended)

orc login

This will open your browser. Sign in, go to your Account page, and generate a token.

Method 2: Direct API Key

orc login --apikey YOUR_API_KEY

Your First Analysis

Step 3: Index Your Codebase

cd your-project
orc index

This creates a searchable database of your code.

Step 4: Start AI Chat

orc

Now you can ask questions about your codebase!

Try: "Show me the most complex functions"

Web Interface

Step 5: Access Dashboard

Visit: http://127.0.0.1:5000

The web interface provides a visual way to explore your code, run analyses, and manage projects.

Quick Commands

  • orc analyze - Run full codebase analysis
  • orc dead - Find potentially unused code
  • orc complexity - Show complex functions
  • orc hotspots - Find problem areas
  • orc status - Check authentication status
