Dependency Graph Visualization

Visual representation of module dependencies in your codebase

Graph Information

This graph shows the dependencies between modules in your codebase:

  • Nodes represent modules/files
  • Edges represent import dependencies
  • Node size is proportional to the number of functions in the module
  • Node color indicates module complexity (blue for high, green for low)
